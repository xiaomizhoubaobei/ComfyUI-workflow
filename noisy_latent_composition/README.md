# 噪声潜在组合示例

您可以在 [ComfyUI](https://github.com/comfyanonymous/ComfyUI) 中加载这些图像以获取完整的工作流。

以下是噪声潜在组合的示例。噪声潜在组合是指在图像完全去噪之前，将潜在图像在仍然处于噪声状态时组合在一起。由于像姿势和主体这样的一般形状在最初的采样步骤中被去噪，这使我们可以例如将具有特定姿势的主体放置在图像的任何位置，同时保持很大程度的一致性。

这是一个示例。此示例包含 4 个组合在一起的图像。1 个背景图像和 3 个主体。
总步数为 16。潜在图像使用不同的提示采样 4 步。背景为 1920x1088，每个主体为 384x768。在这 4 步之后，图像仍然非常嘈杂。然后将主体组合（粘贴）到背景上，并应用一些羽化效果。然后在这个组合图像上运行剩余的采样步骤。


这些示例是使用 WD1.5 beta 3 illusion 模型完成的。

您可以看到，从不同噪声潜在图像组合在一起的主体的确相互交互，因为我在提示中放置了"holding hands"（手牵手）。您还会注意到背景是多么一致，这显示了这种方法是多么强大。

这种技术有一些局限性，例如它无法控制主体上的细节，如眼睛颜色，但在主体位置、姿势和一般颜色方面似乎非常有效。
