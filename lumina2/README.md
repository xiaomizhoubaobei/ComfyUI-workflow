# Lumina Image 2.0

[Lumina Image 2.0](https://github.com/Alpha-VLLM/Lumina-Image-2.0) 是一个有趣的扩散模型，它使用 gemma 2 2B 作为其文本编码器。

## 基本工作流

下载 [lumina_2.safetensors](https://huggingface.co/Comfy-Org/Lumina_Image_2.0_Repackaged/blob/main/all_in_one/lumina_2.safetensors) 并将其放入您的 ComfyUI/models/checkpoints 目录。

然后您可以在 ComfyUI 中加载或拖动以下图像以获取工作流：
